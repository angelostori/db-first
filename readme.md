# DB Schema
Modellizzare la struttura di una tabella per memorizzare tutti i dati riguardanti delle auto usate messe in vendita da un concessionario

## Table name: cars (Entity name: car)

Columns:

- id
- car_brand
- car_model
- fuel_type
- fuel_consumption
- km_driven
- year
- price
- color
- transmission
- description
- car_horsepower
- displacement(cc)
- places
- doors
- marches
- empty_weight
- emission_class
- type_of_vehicle
- traction
- previous_owners
